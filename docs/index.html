Orbit

Manage development environment port forwarding on macOS

Download Latest View on GitHub

Features

  • Define environments with loopback interface aliases
  • Configure services with kubectl, ssh tunnels, cloudflared, etc.
  • Toggle environments from menubar or main window
  • Variable substitution ($IP, $IP2, etc.)
  • Import/export environment configurations
  • Automatic updates via Sparkle

Requires macOS 13.0 or later

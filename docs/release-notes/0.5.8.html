What's New in Orbit 0.5.8

Bug Fixes

  • Fixed variable substitution where $IP could incorrectly match inside $IP2, $IP3, etc.
  • Fixed race condition when quitting during environment activation
  • Fixed process PATH being overwritten instead of preserved from user's shell
  • Fixed config backup restoring on I/O errors (now only restores on corruption)
  • Fixed import/editor inconsistency for 127.0.0.1 validation

Improvements

  • Improved SwiftUI performance by decoupling log storage from environment state
  • Automatic cleanup of stale loopback aliases on app launch
  • Automatic config backup before each save
  • Standardized logging across the codebase

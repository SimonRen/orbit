What's New in Orbit 0.5.5

  • Domain Aliases for Interfaces: Add optional domain patterns (e.g., *.meera-dev) to interfaces for documentation. Useful when using dnsmasq for local DNS resolution.
  • Improved Copy for AI: The "Copy for AI" output now includes domain alias information with explanatory context.
  • Backward Compatible Exports: Environment exports remain compatible with older versions when no domains are configured.

What's New in Orbit 0.5.13

  • Fuzzy search for namespaces and services in K8s import (e.g. "abc123" matches "abc-def-123")
  • Fixed shell injection risk with context/service names containing spaces or special characters
  • Fixed thread safety issue in kubectl output handling
  • Fixed stale data shown when namespace/service fetch fails
  • Cancelled K8s fetches now properly terminate kubectl processes
  • Fixed orb-kubectl being selectable when not installed
